## Added

- Draft RFC for the Argus C2 Protocol defining a real-time command-and-control bus for ACP-integrated workflows.
- Expanded Argus C2 draft with explicit bootstrapping, acknowledgement, and security incident guidance.
- Introduced the Quantum Gravity Encryption (QGE) RFC plus an accompanying example envelope for encrypted checkout payloads.
- Added a JSON Schema for the QGE envelope along with documentation references so integrators can validate payloads.
